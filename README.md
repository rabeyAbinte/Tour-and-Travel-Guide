# Tour & Travel Guide

A simple full-stack web application for booking tours and hotels.  
Frontend: HTML, CSS, JavaScript  
Backend: Node.js, Express, MongoDB

---

## 🚀 Features
- User Registration & Login
- Places & Hotels listing
- Booking system
- Dark theme responsive UI
